# 🚀 Space Odyssey — "What I Love" Project

A fully responsive multi-page website about space exploration, built with **HTML & CSS only**.

## 📁 Project Structure

```
what-i-love-project/
├── index.html
├── about.html
├── gallery.html
├── contact.html
│
├── style.css
│
├── /images
│   └── image files here
│
└── /fonts or /icons (optional)
```

## 🚀 Getting Started

1. Open `index.html` in your browser — no server needed
2. All pages link to each other via the sticky navigation bar
3. No JavaScript — pure HTML & CSS

## 📤 Deployment (GitHub Pages)

1. Push this folder to a public GitHub repo
2. Go to **Settings → Pages → Deploy from main branch**
3. Live at: `https://yourusername.github.io/what-i-love-project`
